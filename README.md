##Unofficial Bearmer Template for Strathclyde University of Glasgow##


Put it into your `TEXINPUTS` path, then use the following to utilize it in your
latex-beamer code:

    \documentclass{beamer}
    \usetheme{STR}

## Options ##

Defaults are 4:3 ratio.
